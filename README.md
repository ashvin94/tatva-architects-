🏗️ Tatva Architects & Realtors

A modern and responsive website for Tatva Architects & Realtors, showcasing architectural projects, real estate services, and company details.

---


📌 Features

- 🏠 Beautiful homepage with hero section
- 🧑‍💼 Partner / Team showcase
- 🏗️ Completed & Ongoing Projects section
- 🖼️ Image gallery for projects
- 📍 Business location integration
- 📱 Fully responsive design (mobile + desktop)
- 🔐 Admin panel (for managing projects - upcoming feature)

---

🛠️ Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
- Supabase (for backend - planned)
